## 杠杆分析

| 维度 | 评估 |
|------|------|
| 杠杆类型 | 代码 / 媒体 / 资本 / 劳动力 / 组合 |
| 杠杆强度 | 1-10分 |
| 叠加潜力 | 代码+媒体 / 媒体+资本 / 无 |
| 可规模化程度 | 高 / 中 / 低 |

## 非对称回报分析

| 维度 | 估算 |
|------|------|
| 最佳情况月收益 | 元 |
| 最差情况月损失 | 元 |
| 非对称比 | :1 |
| 收益持续性 | 一次性 / 月度 / 年度 / 被动 |

**判定**：
- ≥ 10:1 → 强Go
- 3:1~10:1 → Go
- < 3:1 → NoGo
