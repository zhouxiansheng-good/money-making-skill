## 工作流配置

### 工作流1: 每日热点内容自动生产

```yaml
name: "每日热点内容自动生产"
trigger: "每天 08:00 自动执行"
steps:
  - step: 1
    name: "抓取热点"
    action: "QClaw web-search"
    input: "搜索今日AI/科技/赚钱领域热搜话题"
    output: "热点话题列表（5-10个）"

  - step: 2
    name: "选题筛选"
    action: "AI分析"
    input: "热点话题列表"
    criteria:
      - "热度 > 1000"
      - "竞争度 < 高"
      - "与账号定位匹配"
    output: "精选选题（3-5个）"

  - step: 3
    name: "内容生成"
    action: "AI生成"
    input: "精选选题"
    template: |
      你是一位{领域}领域的资深创作者。
      请围绕"{选题}"写一篇{字数}字的文章。
      要求：
      1. 标题吸睛，包含数字或疑问
      2. 开头3行抓住注意力
      3. 内容有干货，有具体案例
      4. 结尾有行动引导
      风格：{风格}
      平台：{平台}
    output: "文章初稿"

  - step: 4
    name: "人工审核"
    action: "人工"
    input: "文章初稿"
    check:
      - "内容准确性"
      - "无违规内容"
      - "有个人观点/独特角度"
    output: "审核通过的文章"

  - step: 5
    name: "自动发布"
    action: "QClaw web-browser"
    input: "审核通过的文章"
    platforms:
      - "头条号"
      - "知乎"
      - "公众号"
    output: "发布成功确认"

  - step: 6
    name: "数据追踪"
    action: "QClaw web-search"
    trigger: "每天 20:00"
    input: "已发布文章的阅读量/互动数据"
    output: "数据报告 → 优化选题策略"
```

### 工作流2: 竞品内容分析+改写

```yaml
name: "竞品内容分析+改写"
trigger: "每周一 10:00 自动执行"
steps:
  - step: 1
    name: "抓取竞品爆款"
    action: "QClaw web-search"
    input: "搜索同领域阅读量>1万的文章"
    output: "爆款文章列表（10篇）"

  - step: 2
    name: "分析爆款规律"
    action: "AI分析"
    input: "爆款文章列表"
    analyze:
      - "标题模式"
      - "开头结构"
      - "内容框架"
      - "互动引导"
    output: "爆款规律总结"

  - step: 3
    name: "改写生成"
    action: "AI生成"
    input: "爆款规律 + 新选题"
    rule: "借鉴结构，不抄袭内容，加入个人观点"
    output: "原创文章"

  - step: 4
    name: "人工审核+发布"
    action: "人工审核 → QClaw自动发布"
    output: "发布成功"
```

## MVP验证计划

| 天数 | 行动 | 目标 |
|------|------|------|
| 第1-2天 | 配置QClaw + 安装技能 | QClaw能自动搜索热点 |
| 第3-5天 | 搭建内容生成工作流 | 每天自动产出3-5篇内容 |
| 第6-10天 | 接入自动发布 | 内容自动发布到平台 |
| 第11-14天 | 观察数据，优化策略 | 至少1篇阅读量>100 |

**成功标准**：14天内至少1篇内容阅读量>100

## 注意事项

- 内容审核必须有人工环节，避免违规
- 初期建议半自动（AI生成+人工审核+手动发布），验证后再全自动
- 不要贪多，先做1个平台做透，再扩展
- 每天抽15分钟看数据反馈，优化选题方向
- 保持个人风格和独特观点，避免纯AI味
