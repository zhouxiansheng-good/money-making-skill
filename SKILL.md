# 赚钱决策Skill

> **财富 = 特定知识 × 杠杆 × 判断力 × 责任感** — Naval Ravikant

## 触发条件

当用户说"赚钱决策"/"评估机会"/"这个值得做吗"/"怎么赚钱"/"赚钱分析"时激活。

## 执行流程

```
用户输入机会描述
  ↓
Layer 1: 思维层
  ├─ 识别该机会涉及的杠杆类型
  ├─ 检查是否匹配你的特定知识
  └─ 运行决策协议
  ↓
Layer 2: 评估层
  ├─ 计算非对称回报比（收益:风险 ≥ 3:1？）
  ├─ 匹配特定知识（匹配度 ≥ 60%？）
  └─ 红线检查（合法？时间可控？资金可控？非纯卖劳？）
  ↓
  ├─ 任一红线不通过 → NoGo + 原因
  ├─ 非对称回报 < 3:1 → NoGo + 建议
  └─ 全部通过 → Go
  ↓
Layer 3: 杠杆层
  ├─ 选择最优杠杆组合
  ├─ 设计AI放大策略
  └─ 评估杠杆叠加效果
  ↓
Layer 4: 行动层
  ├─ 设计MVP验证实验
  ├─ 制定2周验证计划
  └─ 定义成功标准
  ↓
输出: Go/NoGo决策 + 杠杆策略 + 行动路径 + MVP计划
```

## 红线规则

以下情况直接NoGo，不进入评估：

| 红线 | 检查内容 |
|------|----------|
| 合法合规 | 是否涉及法律灰色地带？ |
| 时间可控 | 每天投入是否超过2小时？ |
| 资金可控 | 启动资金是否超过500元？ |
| 非纯卖劳 | 是否纯时间换钱、无可规模化空间？ |

## 杠杆层级参考

| 层级 | 杠杆类型 | 扩展性 | 适合场景 |
|------|----------|--------|----------|
| 1（最强） | 代码 | 无限 | 软件产品、工具、SaaS |
| 2 | 媒体 | 极高 | 内容创作、自媒体、课程 |
| 3 | 资本 | 高 | 投资、资产配置 |
| 4（最弱） | 劳动力 | 有限 | 团队管理、外包 |
