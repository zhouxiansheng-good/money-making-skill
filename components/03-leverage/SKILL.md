# 03-leverage：杠杆层

> **"杠杆决定收入天花板"** — 光有特定知识还不够，需要杠杆把它放大。

## 职责

选择最优杠杆组合，设计AI放大策略，评估杠杆叠加效果。核心原则：**优先代码杠杆和媒体杠杆，叠加使用效果最强**。

## 前置条件

- 评估层（02-evaluate）已输出Go决策
- 杠杆类型和强度已识别

## 组件

### 1. 代码杠杆引擎

**目标**：用AI构建可规模化的软件产品。

**适用条件**：
- 你有技术开发能力 ✅
- 机会可以产品化为软件/工具
- 边际成本接近零

**代码杠杆路径**：

```
Level 1: 脚本/工具（1-3天）
  - 用AI写Python/Shell脚本解决特定问题
  - 在闲鱼/淘宝出售脚本服务
  - 单价: 50-500元/个

Level 2: 小工具/插件（1-2周）
  - 用AI构建浏览器插件/VS Code插件/命令行工具
  - 上架到插件商店/产品平台
  - 单价: 免费+付费高级版

Level 3: SaaS产品（1-3月）
  - 用AI构建完整的Web应用
  - 订阅制收费
  - 单价: 29-299元/月
```

**AI辅助开发策略**：
- 用AI生成代码框架和核心逻辑
- 用AI写测试用例
- 用AI做代码审查
- 用AI生成文档和README

### 2. 媒体杠杆引擎

**目标**：用AI批量生产内容，实现创作一次、触达百万人。

**适用条件**：
- 你有内容创作意愿
- 机会可以通过内容传播
- 可以建立个人品牌

**媒体杠杆路径**：

```
Level 1: 自媒体文章（每天1-2小时）
  - 用AI生成文章初稿，人工优化
  - 发布到头条号/知乎/公众号
  - 收入: 广告分成 500-5000元/月

Level 2: 短视频内容（每天1-2小时）
  - 用AI写脚本+生成数字人视频
  - 发布到抖音/快手/小红书
  - 收入: 流量收益+带货 1000-10000元/月

Level 3: 知识产品（积累后）
  - 将内容体系化为课程/电子书
  - 通过私域/平台销售
  - 收入: 被动收入 2000-20000元/月
```

**AI内容生产流水线**：

```
选题 → AI调研热点 → AI生成大纲 → AI填充内容 → 人工审核优化 → 发布
  ↑                                                    ↓
  └──────── 数据反馈（阅读量/互动/收益）←───────────────┘
```

### 3. 杠杆叠加器

**目标**：组合多种杠杆，实现1+1>2的效果。

**叠加策略**：

| 叠加组合 | 效果 | 操作路径 |
|----------|------|----------|
| 代码+媒体 | 最强叠加 | 做产品+用内容推广产品 |
| 媒体+资本 | 中等叠加 | 做内容+投资收益复投 |
| 代码+资本 | 中等叠加 | 做产品+融资/投资 |
| 三重叠加 | 最强但最复杂 | 做产品+内容推广+资本放大 |

**叠加实施顺序**：

```
Step 1: 先启动单一杠杆（媒体杠杆最容易启动）
Step 2: 积累受众和现金流
Step 3: 叠加代码杠杆（将知识产品化）
Step 4: 叠加资本杠杆（用收益投资）
```

## 杠杆选择决策树

```
你有技术开发能力？
  ├─ 是 → 机会可产品化？
  │       ├─ 是 → 代码杠杆（优先）+ 媒体杠杆（推广）
  │       └─ 否 → 媒体杠杆（优先）
  └─ 否 → 你有内容创作意愿？
          ├─ 是 → 媒体杠杆（优先）
          └─ 否 → 资本杠杆（需本金）或 劳动力杠杆（不推荐）
```

- 杠杆策略报告（最优杠杆类型+叠加建议）
- AI放大执行计划
- 杠杆强度评分
