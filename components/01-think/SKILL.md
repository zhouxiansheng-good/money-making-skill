# 01-think：思维层

> **"没有调查，没有发言权"** — 先理解自己，再判断机会。

## 职责

帮用户建立赚钱的底层思维框架：发现特定知识、识别杠杆、系统化决策、追踪复利。

## 组件

### 1. 特定知识发现器

**目标**：找到你的特定知识 — 别人难以复制的、你独有的技能组合。

**发现流程**：

```
Step 1: 回答4个自问问题
  - 什么事情你做起来像在玩，别人做起来像在受苦？
  - 什么话题你能聊3个小时不带停的？
  - 别人经常夸你什么，但你觉得"这有什么难的"？
  - 你有什么技能是学校里学不到的？

Step 2: 列出技能/兴趣/经历清单
  - 技能：你会什么？（编程
Step 2: 回答4个核心问题
  Q1: 什么事情你做起来像在玩，别人做起来像在受苦？
  Q2: 什么话题你能聊3个小时不带停的？
  Q3: 别人经常夸你什么，但你觉得"这有什么难的"？
  Q4: 你有什么技能是学校里学不到的？

Step 3: 交叉验证
  - 将4个问题的答案取交集
  - 交集区域 = 你的特定知识候选区

Step 4: 市场验证
  - 候选特定知识是否有人愿意付费？
  - 是否有竞品？竞品是否容易复制？
  - 是否能叠加杠杆（代码/媒体/资本）？

Step 5: 输出特定知识画像
```

**特定知识公式**：`特定知识 = (天赋 + 真正的痴迷 + 深度练习) × 独特的人生经历`

**反面特征**（不是特定知识）：
- 可以通过培训学会的技能（Excel、PPT、开车）
- 标准化的职业能力（会计、文员）
- 单一维度的通用技能

**正面特征**（是特定知识）：
- 跨领域的独特组合（金融+编程+用户心理）
- 行业深耕10年+的直觉和人脉
- 天赋驱动的自然优势

### 2. 杠杆识别器

**目标**：判断一个机会用什么杠杆，评估杠杆强度。

**识别流程**：

```
输入: 机会描述
  ↓
Step 1: 判断杠杆类型
  - 能否做成软件/工具？ → 代码杠杆（强度10）
  - 能否做成内容/课程？ → 媒体杠杆（强度8）
  - 需要投资/资产配置？ → 资本杠杆（强度6，需本金）
  - 需要雇人/管理团队？ → 劳动力杠杆（强度3，管理成本高）

Step 2: 评估杠杆叠加潜力
  - 代码+媒体 → 强叠加（产品+传播）
  - 媒体+资本 → 中叠加（影响力+投资）
  - 代码+资本 → 中叠加（产品+融资）
  - 单一杠杆 → 无叠加

Step 3: 输出杠杆评估
  - 最优杠杆类型
  - 杠杆强度评分（1-10）
  - 叠加建议
```

**杠杆层级排序**（从强到弱）：

| 层级 | 杠杆 | 扩展性 | 边际成本 | 适合谁 |
|------|------|--------|----------|--------|
| 1 | 代码 | 无限 | ≈0 | 有技术开发能力 |
| 2 | 媒体 | 极高 | ≈0 | 有内容创作能力 |
| 3 | 资本 | 高 | 低 | 有本金 |
| 4 | 劳动力 | 有限 | 高 | 有管理能力 |

### 3. 决策协议

**目标**：系统化决策流程，避免凭感觉做判断。

**7步决策序列**：

```
Step 1: 遗憾最小化
  "80岁时我会后悔没做这件事吗？"
  → 不会有长期遗憾 → 大概率可以跳过
  → 会有长期遗憾 → 继续评估

Step 2: 可逆性测试
  "这个决定能撤回吗？"
  → 能撤回 → 快速决定，立刻执行
  → 不能撤回 → 慢慢想，想清楚再动手

Step 3: 收益/风险比
  "如果一切顺利 vs 一切糟糕，比例是多少？"
  → ≥ 10:1 → 强Go
  → 3:1~10:1 → Go
  → < 3:1 → NoGo或重新设计

Step 4: 杠杆倍增器
  "这件事会让我的杠杆增加还是减少？"
  → 增加杠杆 → 继续
  → 减少杠杆 → NoGo

Step 5: 选择权检查
  "这件事会打开更多门还是关上门？"
  → 打开门 → 继续
  → 关门 → 谨慎

Step 6: 真实性过滤
  "我做这件事是因为我想做，还是因为别人都在做？"
  → 内在驱动 → 继续
  → 社会压力 → NoGo

Step 7: 切身利益
  "我正在冒什么无法收回的风险？"
  → 时间是最不可替代的资产
  → 只投入你能承受失去的
```

### 4. 复利追踪器

**目标**：追踪知识、关系、声誉、技能的复利积累。

**四维复利模型**：

| 维度 | 每日行动 | 复利效应 | 追踪指标 |
|------|----------|----------|----------|
| 知识复利 | 读书1小时 | 认知远超同龄人 | 本月读书量+关键洞察 |
| 关系复利 | 帮助1个人 | 10年后人脉强到难以想象 | 本月深度连接数 |
| 声誉复利 | 做好1件事 | 声誉是最深的护城河 | 本月口碑事件数 |
| 技能复利 | 刻意练习1项 | 特定知识持续深化 | 本月技能提升证据 |

**复利公式**：`每日进步1% → 一年后变强37倍`

**反噬警告**：`每日退步1% → 一年后只剩3%`

## 产出

- 特定知识画像
- 杠杆评估报告
- 决策协议执行结果（Go/NoGo+理由）
- 复利追踪记录

## 路由

- 特定知识+杠杆评估完成 → 进入 `components/02-evaluate/SKILL.md`
- 决策协议结果为NoGo → 终止，告知原因
- 需要更多自我探索 → 回到特定知识发现器
